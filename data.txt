COMPUTER BASICS:
A computer is an electronic device used to store and process data.
A computer system consists of hardware and software.

HARDWARE:
The CPU is the brain of the computer and performs calculations.
RAM is temporary memory used by the computer for fast access.
A hard drive or SSD stores files and programs permanently.
A GPU handles graphics and accelerates visual tasks.
The motherboard connects all computer components.
An input device allows users to enter data into a computer. Examples include keyboard and mouse.
An output device displays results. Examples include monitor and printer.
A power supply unit provides electricity to computer components.

OPERATING SYSTEMS:
An operating system manages hardware and software resources.
Windows, macOS, and Linux are popular operating systems.
The operating system provides a user interface such as GUI or CLI.

SOFTWARE:
Software is a collection of programs that tells a computer what to do.
Application software includes browsers, word processors, and games.
System software supports computer operations, such as drivers and OS tools.

NETWORKING:
A network is a group of connected devices that share data.
The internet is a global network of networks.
A router connects multiple devices to a network.
An IP address identifies a device on a network.
Wi-Fi is a wireless networking technology.

PROGRAMMING:
Programming is writing instructions for a computer to follow.
Python is a popular beginner-friendly programming language.
JavaScript is used for web development.
HTML and CSS are used to build web pages.
A variable stores data in a program.
A loop repeats instructions until a condition is met.

CYBERSECURITY:
A firewall protects a device or network from unauthorized access.
Antivirus software detects and removes malware.
A strong password increases security.
Phishing is a scam used to steal information.

DATABASES:
A database stores and organizes data electronically.
SQL is a language used to manage databases.

CLOUD COMPUTING:
Cloud computing means storing and accessing data on the internet instead of your device.
Examples include Google Drive, Dropbox, and iCloud.

ARTIFICIAL INTELLIGENCE:
Artificial Intelligence allows computers to learn and make decisions.
Machine learning is a type of AI that learns from data.

TROUBLESHOOTING:
Restarting a computer can fix many basic problems.
Slow performance may be caused by low RAM or too many background apps.
